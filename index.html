JetBrains Upsource Early Access Program

Thank you for interest in Upsource, a new repository browser and code review tool from JetBrains.

If you would like to try a pre-release build of Upsource, please fill out the form below. By providing us with an overview of your development environment, you will help us make better decisions as we plan further development of Upsource.

Please understand that since this is pre-release software, there's no guarantee that a EAP build will even run in your environment.

That said, we put our best effort into testing EAP builds and we will be happy to work with you to resolve any issues encountered during your participation in Upsource Early Access Program.

Until Upsource hits final release, it is free to use and doesn't require a license key.





Unfortunately Upsource only supports Git, Mercurial, Subversion and Perforce. If you're exclusively using a version control system other than the 4 supported systems, you will not be able to use Upsource at all.

Please note that you will only be able to use Upsource with Git, Mercurial, Subversion and Perforce repositories.
